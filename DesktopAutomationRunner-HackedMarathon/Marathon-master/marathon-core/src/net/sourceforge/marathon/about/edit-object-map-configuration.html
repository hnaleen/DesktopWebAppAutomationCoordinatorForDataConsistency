Edit Object Map Configuration (only on MarathonITE)

Based on Marathon, Jalian Systems provides a commercial version with extra features and support. You can use MarathonITE and reduce the effort required for creating resilient test suites. Use the included extract method refactoring to reuse your test scripts. Perform exploratory or semi-automatic testing through MarathonITE and share the test scripts with the development team.

While recording test scripts Marathon creates object map entries. Each entry has a name and corresponding recognition properties. Marathon uses object map configuration to identify the recognition properties and name a component. The object map configuration is stored in a YAML formatted text file - omap-configuration.yaml in the project folder. This file can be modified using any text editor.

MarathonITE includes a configuration editor for the omap-configuration.yaml file. This reduces the chances of inadvertently corrupting the file.
