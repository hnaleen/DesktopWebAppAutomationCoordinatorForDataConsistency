Create a Data Driven Test (only on MarathonITE)

Based on Marathon, Jalian Systems provides a commercial version with extra features and support. You can use MarathonITE and reduce the effort required for creating resilient test suites. Use the included extract method refactoring to reuse your test scripts. Perform exploratory or semi-automatic testing through MarathonITE and share the test scripts with the development team.

MarathonITE supports Data Driven Testing by which you can execute the same test script for multiple inputs. The advantage of data driven testing is the ease with which you can add multiple inputs when multiple test cases are identified.
